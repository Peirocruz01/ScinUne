

# ScinUne — Premium Skincare Brand Homepage

A biology-driven skincare brand website with a clean, scientific, and luxurious aesthetic. Deep botanical greens, warm beige backgrounds, glass-style UI overlays, and elegant typography.

---

## Design System Setup
- Color palette: deep botanical green (#0F3D2E), forest green (#1F5A45), soft gold (#C9A227), warm beige (#F3EFE7)
- Typography: Plus Jakarta Sans with defined heading/body scale
- Spacing tokens, border radius (32px containers, 20px cards, pill buttons), soft shadows, and glass effects
- Reusable components: SectionWrapper, ProductCard, FeatureCard, GlassDataCard

## Section 1 — Hero (Dark Botanical Background)
- Full-width dark green gradient with large rounded edges
- Bold H1: "Test better. Treat better." with subtext
- Two pill CTA buttons side by side
- Layered composition: skincare face image, floating glass UI cards (circular data chart, score card), and a serum bottle — overlapping for depth
- Subtle fade-in and float animations on the glass cards

## Section 2 — "From Biology to a Better Routine"
- Continues dark green background
- Pill badge label above centered H2
- 3-column card grid:
  - Card 1: Skincare close-up image + "Capture the hidden signals" text
  - Card 2: Mock data UI card with scores and metrics
  - Card 3: Skincare application image + "Craft a responsive strategy" text
- Each card with rounded corners, soft shadow, and "Included in Bioflora" badge

## Section 3 — "Your Skin Story Revealed" (Light Background)
- Switches to warm beige background
- Two-column layout:
  - Left: Badge + H2 title + descriptive paragraph
  - Right: Dark green card with 3D face analysis visualization, vertical numbered navigation (01–05), and subtle bottom UI panel

## Section 4 — "Four Steps. One Ecosystem."
- Beige background continues
- Badge + left-aligned H2 + subtitle
- Horizontal scrollable product card grid (4 cards):
  - Each card: product bottle image, step label, title, description, ingredient tag badges
- Navigation arrows (prev/next) on the right side
- Dot indicators below

## Section 5 — Footer
- Dark green background
- 4 columns: Logo + description, Shop links, Learn links, Newsletter input + subscribe button
- Social media icons row
- Clean, minimal layout

## Interactions & Polish
- Button hover transitions (200ms ease)
- Card hover lift (2–4px) with shadow increase
- Hero floating elements: subtle float animation
- Product carousel: arrow navigation with smooth scroll
- Mobile-first responsive: stacked columns, reduced heading sizes, accessible focus rings

## Images
- High-quality Unsplash skincare images (face close-ups, botanical backgrounds, serum bottles)
- Lucide icons throughout (ArrowRight, ChevronLeft, ChevronRight, Check, Leaf, Droplet)
